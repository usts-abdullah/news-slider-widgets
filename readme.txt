WP News Slider Widgets 

== Description ==

"WP News Slider Widgets" plugin is a News or image slider widget.
it gives custom widget functionality to design widget for footer or any widget area to make news/image slider for your wordpress site. 
you can add/edit/delete news and manage which news to show.

You will be able to:

	
= KEY BENEFITS: =
- create custom widget for news slider. 
- you can give different colors in widget.
- all text and captions are dynamic. you set it from admin.
- can add image in news slider.
- can add/edit/delete and active/enactive news.
- rich color picker are used in admin to design widgets.

== Installation ==

1) Copy/Upload 'gen-wp-news-slider-widgets' folder to the '/wp-content/plugins/' directory
2) Activate 'WP News Slider Widgets' from wp plugin option in admin area
3) Plugin will appear in the menu bar of WP Dashboard
4) add News Slider widget from your admin->Widgets area to your expected footer or widgets area.
5) Design and setup your widget 
6) add news from admin->news slider-> add news
new you should see news slider in your site front end.
